AD NETWORKS

Setting Up AD NETOWRKS

You can choose to put the ad networks in any of the 3 screens: MainMenu.m , ViewController.m , GameOver.m 

Now I will show you how to use the builder in order to implement any of the 5 AD NETWORKS in that way that you can use that builder in any of your games. For more information and video tutorials please click here: http://codecanyon.net/item/mobile-builder-ios/8432703?ref=vikt0r40

ChartBoost

In order to implement Chartboost in the game open one of the 3 screens that I mention earlier. Find the method ViewDidLoad it looks like that: 

Immediately after [super viewDidLoad]; write this: 

[[MobileBuilder startBuilding]initialiseChartboost:@"Chartboost ID" appSignute:@"App Signiture"];

[[MobileBuilder startBuilding]chartboostShowAd];

 

Instead of Chartboost ID place your ID and instead of App signiture place the signite you will get from Chartboost.com

 

When you finish it needs to look like this: 

This is all you need to do. Now when you run the game and go to the selected screen chartboost will pop.

RevMob

RevMob is the same as chartboost but the methods are: 

[[MobileBuilder startBuilding]initialiseRevMob:@"ID" testMode:YES];

[[MobileBuilder startBuilding]revmobShowAd];

 

After finish it needs to look like this: 

AdMob

AdMob is even easier again in ViewDidLoad just place this line: 

[[MobileBuilder startBuilding]admobStart:@"HERE THE ADMOB ID" controller:self];

iAD

iAD is the same as AdMob add this line: 

[[MobileBuilder startBuilding]iAdStart:self];

 

Note here you don't even need a id.

AppLovin

Add these lines in ViewDidLoad: 

    [[MobileBuilder startBuilding]initialiseAppLovin];

    [[MobileBuilder startBuilding]appLovinShowInterstitial];

In-App Purchases

In order to use in-app purchases you need to have active iTunes Account. The In-app purchases will remove the ads. 

IT IS REALLY IMPORTANT TO FOLLOW THESE INSTRUCTIONS LINE BY LINE.

 

1. To connect the in-app purchases with the ads you need to wrap the code that you add in the ViewDidLoad methods with if statement. For example: 

- If you added in ViewDidLoad method to show chartboost: 

 

- You need to make it like that: 

    if([[NSUserDefaults standardUserDefaults]boolForKey:@"isPurchase"] == NO) {

        [[MobileBuilder startBuilding]initialiseChartboost:@"213123" appSignute:@"s123c123c12"];

        [[MobileBuilder startBuilding]chartboostShowAd];
